# 广州100

2019年广州100越野赛60公里组别，完成时间11小时16分54秒。

## 比赛感受

这是一场非常棒的越野赛，赛道穿越广州周边山区，组织有序，风景优美。成功完成了比赛目标。

## 比赛经历

比赛当天天气适宜，全程保持了稳定的配速，虽然距离较长，爬升较大，但依靠良好的训练基础坚持到了终点。
